# flutter_kit_dio

## 说明

主要用于对网络请求（基于`dio`库）的打印，可以在 Android Studio 和 APP 内查看相关的信息。

打印的内容包括：
+ 时间
+ 请求返回的状态码
+ 请求方式
+ 请求时长
+ 请求连接
+ 请求头
+ 请求参数
+ 响应头
+ 响应内容

另外对于内容部分，提供了搜索文本高亮的功能，能帮助我们快速定位我们想要的内容。
